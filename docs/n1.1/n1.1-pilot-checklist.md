# N1.1 Pilot Checklist (10-20 learners)

Track:
- completion rate
- route distribution (A/B/C)
- shadow pass rates by route
- attempts-to-secure
- intervention rate
- median time-to-status

PASS gate:
- >=90% diagnostic completion
- >=70% Developing+
- Route C resolves m1/m2 in <=2 loops for >=60%
- intervention <=20%
