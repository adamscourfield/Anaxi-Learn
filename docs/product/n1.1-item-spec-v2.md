# N1.1 Item Spec v2 (Place Value)

## Objective
Production-ready N1.1 item design using best-fit question types.

## Diagnostic bundle (4) — FINAL

1. **N1.1-D1 Anchor (SHORT_NUMERIC)**
   - Prompt: `What is the value of the digit 7 in 3,742,915?`
   - Correct answer: `700000`
   - Metadata: `question_role=anchor`, `misconception_tag=pv_m1_place_vs_value`, `transfer_level=none`, `strictness_level=normalized`

2. **N1.1-D2 Misconception probe (MCQ)**
   - Prompt: `In 504,081, what is the place value of digit 4?`
   - Choices: `thousands`, `ten-thousands`, `hundreds`, `ones`
   - Correct answer: `thousands`
   - Metadata: `question_role=misconception_probe`, `misconception_tag=pv_m2_zero_shift`, `transfer_level=none`, `strictness_level=exact`

3. **N1.1-D3 Structure probe (SHORT_TEXT)**
   - Prompt: `Write 8,030,406 in expanded form.`
   - Correct answer (canonical): `8000000 + 30000 + 400 + 6`
   - Metadata: `question_role=prerequisite_probe`, `misconception_tag=pv_m2_zero_shift`, `transfer_level=low`, `strictness_level=normalized`

4. **N1.1-D4 Transfer check (MCQ)**
   - Prompt: `Which number has digit 5 worth 50,000?`
   - Choices: `5,203,410`, `1,052,340`, `305,214`, `250,431`
   - Correct answer: `1,052,340`
   - Metadata: `question_role=transfer`, `misconception_tag=pv_m3_reading_direction`, `transfer_level=medium`, `strictness_level=exact`

## Route-specific shadows (6)
- **Route A shadows (2):**
  - one short numeric
  - one place-name short entry
- **Route B shadows (2):**
  - expanded form entry
  - interpretation MCQ
- **Route C shadows (2):**
  - error-correction short entry
  - misconception-disambiguation MCQ

## Mastery gate requirement
- Must pass 2/2 in selected route
- Must pass at least 1 non-MCQ item in gate cycle
- Secure only when transfer signal is also positive

## Marking tolerance rules
- Numeric entry normalization:
  - trim spaces
  - ignore commas
  - normalize leading zeros
- Text entry normalization:
  - case-insensitive
  - alias map (e.g., "thousands" == "thousand place")

## Data tagging (required)
Per item include:
- question_role
- misconception_tag
- route (if shadow)
- transfer_level
- answer_type
- strictness_level (exact / normalized)
