## Mastery gate requirement
- Must pass 2/2 in selected route
- Must pass at least 1 non-MCQ item in gate cycle
- Secure only when transfer signal is also positive

### Non-MCQ gate coverage (explicit)
- Route A gate includes **2 non-MCQ** items (`N1.1-S1`, `N1.1-S2`)
- Route B gate includes **1 non-MCQ** item (`N1.1-S3`) + 1 MCQ (`N1.1-S4`)
- Route C gate includes **1 non-MCQ** item (`N1.1-S5`) + 1 MCQ (`N1.1-S6`)

## Marking tolerance rules
- Numeric entry normalization:
  - trim spaces
  - ignore commas
  - normalize leading zeros
- Text entry normalization:
  - case-insensitive
  - alias map (e.g., "thousands" == "thousand place")

## Data tagging (required)
Per item include:
- question_role
- misconception_tag
- route (if shadow)
- transfer_level
- answer_type
- strictness_level (exact / normalized)
