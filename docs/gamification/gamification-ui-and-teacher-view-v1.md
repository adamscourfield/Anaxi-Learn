# Gamification UI + Teacher View (v1)

## Student UI blocks
1. **Top HUD**
   - XP bar
   - current streak
   - token count
2. **Question panel**
   - prompt + input
   - confidence hint
3. **Feedback rail**
   - instant correctness
   - skill status progress (Not Yet/Developing/Secure)
4. **Quest panel**
   - 1-2 active micro-goals

## Session-end card
- Questions attempted
- Accuracy
- XP earned
- Tokens earned
- Skill transitions achieved
- "Next mission" CTA

## Teacher view (minimal useful)
- Class engagement heatmap:
  - completion rate
  - avg attempts-to-secure
  - route success rates
- Student drilldown:
  - misconception pattern
  - reward events timeline
  - intervention flags

## Teacher controls
- Toggle cosmetic intensity (low/normal/high)
- Disable streak pressure messaging
- Highlight resilience rewards in feedback

## Analytics essentials
- Engagement: active minutes, return rate, completion
- Learning quality: shadow pass, secure transitions, intervention rate
- Motivation health: streak retention, reward claim rate, retry recovery rate
